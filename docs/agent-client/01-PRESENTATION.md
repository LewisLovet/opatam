# Opatam - Presentation generale

---

## Qu'est-ce qu'Opatam ?

Opatam est une plateforme de reservation en ligne multi-secteurs qui met en relation les clients avec des prestataires de services partout en France. Que ce soit pour un rendez-vous chez le coiffeur, un massage, un coaching sportif, un cours de musique ou un artisan, Opatam permet de trouver le bon professionnel et de reserver en quelques secondes.

---

## Proposition de valeur

### Pour les clients
- **100% gratuit** : aucun frais de reservation, aucun cout cache
- **Simplicite** : reservation en quelques taps sur mobile ou en quelques clics sur le web
- **Sans compte obligatoire** : sur le site web, il est possible de reserver sans creer de compte
- **Rappels automatiques** : notifications push et emails avant chaque rendez-vous (24h et 2h avant)
- **Multi-secteurs** : plus de 15 categories de services en un seul endroit
- **Disponibilites en temps reel** : visualisation instantanee des creneaux libres

### Pour les prestataires
- **0% de commission** : prix fixe mensuel, sans commission sur les reservations
- **Agenda en ligne 24h/24** : les clients reservent meme quand le prestataire ne peut pas repondre
- **Gestion complete** : services, equipe, lieux, disponibilites, reservations, avis
- **Visibilite** : profil public sur la marketplace avec SEO optimise
- **Rappels automatiques** : les clients sont automatiquement prevenus, reduction des no-shows
- **Application mobile dediee** : gestion du planning depuis le telephone

| Interface | Cible | Disponibilite |

## Technologie

- **Site web** : heberge sur Vercel (haute disponibilite, performances optimales)
- **Application mobile** : developpee avec Expo / React Native (iOS, Android a venir)
- **Backend** : Firebase (Google Cloud) pour l'authentification, la base de donnees et les notifications
- **Emails** : envoyes via un service professionnel (Resend)
- **Paiements** : securises par Stripe (leader mondial du paiement en ligne)
