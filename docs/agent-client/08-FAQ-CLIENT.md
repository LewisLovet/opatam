# FAQ Client

> Questions frequentes des clients qui utilisent Opatam pour reserver des services.

---

## Reservation

### Q : Opatam est-il gratuit pour les clients ?
**R** : Oui, Opatam est 100% gratuit pour les clients. Il n'y a aucun frais de reservation, aucun abonnement, aucun cout cache. Vous ne payez que la prestation directement au prestataire.

### Q : Dois-je creer un compte pour reserver ?
**R** : Sur le **site web**, vous pouvez reserver sans creer de compte. Il suffit de fournir votre nom, email et telephone. Sur l'**application mobile**, un compte est necessaire (inscription rapide avec Google, Apple ou email).

### Q : Comment reserver un rendez-vous ?
**R** : 
1. Trouvez un prestataire via la recherche ou les categories
2. Choisissez un service
3. Selectionnez une date et un creneau horaire
4. Renseignez vos coordonnees (si pas de compte)
5. Confirmez la reservation

### Q : Les creneaux affiches sont-ils en temps reel ?
**R** : Oui, les creneaux sont calcules en temps reel en tenant compte des horaires du prestataire, des reservations existantes et des indisponibilites. Un creneau affiche est un creneau disponible au moment de la consultation.

### Q : Que signifie le statut "En attente" ?
**R** : Certains prestataires preferent confirmer manuellement chaque reservation. Le statut "En attente" signifie que votre demande a ete envoyee et attend la confirmation du prestataire. Vous recevrez une notification des qu'il confirme.

### Q : Puis-je choisir un membre specifique de l'equipe ?
**R** : Oui, si le prestataire a une equipe, vous pouvez choisir le professionnel de votre choix. Vous pouvez aussi selectionner "Peu importe" pour prendre le premier disponible.

## Annulation et modification

### Q : Comment annuler un rendez-vous ?
**R** : Deux options :
- **Sur l'application mobile** : allez dans "Mes RDV", ouvrez le detail du rendez-vous et appuyez sur "Annuler"
- **Par email** : cliquez sur le lien d'annulation dans l'email de confirmation

### Q : Y a-t-il des frais d'annulation ?
**R** : Opatam ne facture aucun frais d'annulation. Cependant, chaque prestataire peut definir ses propres conditions (delai minimum d'annulation avant le rendez-vous). Si le delai est depasse, l'annulation peut ne pas etre possible en ligne.

### Q : Puis-je modifier la date ou l'heure de mon rendez-vous ?
**R** : Seul le prestataire peut modifier l'horaire d'un rendez-vous. Si vous souhaitez changer de creneau, vous pouvez annuler votre rendez-vous actuel et en reserver un nouveau, ou contacter directement le prestataire.

### Q : Que se passe-t-il si le prestataire annule mon rendez-vous ?
**R** : Vous recevrez une notification push et un email vous informant de l'annulation. Vous pourrez alors reserver un nouveau creneau.

## Notifications et rappels

### Q : Vais-je recevoir un rappel avant mon rendez-vous ?
**R** : Oui, vous recevrez automatiquement :
- Un rappel **24 heures avant** votre rendez-vous (push + email)
- Un rappel **2 heures avant** (push + email)
Les rappels ne sont pas envoyes entre 23h et 6h.

### Q : Je ne recois pas les notifications push, que faire ?
**R** : Verifiez que :
1. Les notifications sont autorisees pour Opatam dans les Reglages de votre iPhone (Reglages > Notifications > Opatam)
2. Les notifications sont activees dans l'application (Profil > Parametres de notification)
3. Votre telephone n'est pas en mode "Ne pas deranger"

### Q : Je ne recois pas les emails de confirmation, que faire ?
**R** : Verifiez votre dossier spam/indesirables. Les emails sont envoyes depuis noreply@kamerleontech.com. Ajoutez cette adresse a vos contacts pour eviter le filtrage.

### Q : Puis-je desactiver les rappels ?
**R** : Oui, depuis l'application mobile : Profil > Parametres de notification. Vous pouvez desactiver les rappels, les notifications de confirmation, d'annulation ou de modification individuellement.

## Compte et profil

### Q : Comment creer un compte ?
**R** : Depuis l'application mobile, vous pouvez vous inscrire avec :
- **Apple** : un tap, rapide et securise
- **Google** : un tap avec votre compte Google
- **Email** : saisissez votre nom, email et mot de passe

### Q : J'ai oublie mon mot de passe, que faire ?
**R** : Sur l'ecran de connexion, appuyez sur "Mot de passe oublie". Saisissez votre email et vous recevrez un lien pour reinitialiser votre mot de passe.

### Q : Comment modifier mes informations personnelles ?
**R** : Dans l'application mobile, allez dans l'onglet Profil, puis "Modifier le profil". Vous pouvez changer votre nom, telephone, photo, ville, etc.

### Q : Comment supprimer mon compte ?
**R** : Depuis l'application mobile : Profil > Parametres > Supprimer le compte. La suppression est definitive et toutes vos donnees personnelles seront effacees.

### Q : Mes donnees sont-elles protegees ?
**R** : Oui, Opatam est conforme au RGPD. Vos donnees sont hebergees sur les serveurs Firebase de Google (infrastructure securisee) et ne sont jamais vendues a des tiers. Consultez notre politique de confidentialite sur opatam.com/confidentialite.

## Avis

### Q : Comment laisser un avis ?
**R** : Apres votre rendez-vous, vous recevrez un email vous invitant a laisser un avis. Vous pouvez donner une note (1 a 5 etoiles) et un commentaire optionnel. Vous avez 14 jours pour laisser votre avis.

### Q : Mon commentaire est-il visible publiquement ?
**R** : Votre note en etoiles est publique et contribue a la note moyenne du prestataire. En revanche, votre commentaire ecrit est prive et visible uniquement par le prestataire.

## Problemes techniques

### Q : L'application ne fonctionne pas / plante, que faire ?
**R** : Essayez de :
1. Fermer et rouvrir l'application
2. Verifier votre connexion internet
3. Mettre a jour l'application depuis l'App Store
4. Si le probleme persiste, contactez-nous a contact@opatam.com

### Q : Je ne trouve pas de prestataire pres de chez moi
**R** : Opatam est en phase de lancement et enrichit son reseau de prestataires chaque jour. Si vous ne trouvez pas de prestataire dans votre zone, n'hesitez pas a recommander Opatam a vos prestataires preferes. Vous pouvez aussi elargir la zone de recherche.

### Q : L'application est-elle disponible sur Android ?
**R** : L'application Android est en cours de developpement. En attendant, vous pouvez utiliser le site web opatam.com depuis votre navigateur Android pour reserver. Vous pouvez vous inscrire sur la liste d'attente Android sur notre site.
